Acerca de este proyecto

Este proyecto Full-stack está dedicado al diseño y desarrollo de una página web sobre el cine. La plataforma ofrece información detallada sobre películas, incluyendo tramas, directores, géneros y calificaciones. Con un enfoque en la experiencia del usuario, esta página web permite a los usuarios descubrir y explorar sus películas favoritas de manera intuitiva y visualmente atractiva. El proyecto combina tecnologías modernas de frontend y backend para ofrecer un rendimiento y una funcionalidad óptimos.
